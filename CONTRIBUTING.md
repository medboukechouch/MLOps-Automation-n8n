# GitHub Issue and PR Templates

## Bug Report Template

```markdown
## Description
Une description claire du bug

## À reproduire
1. Aller à ...
2. Cliquer sur ...
3. Voir l'erreur ...

## Comportement attendu
Décrire ce qui devrait se passer

## Capture d'écran
Si applicable

## Environnement
- OS: [ex: Windows 10]
- Python: [ex: 3.9.5]
- Version du projet: [ex: 1.0.0]

## Logs/Erreur
```
Coller l'erreur complète ici
```
```

## Feature Request Template

```markdown
## Description
Description claire de la feature

## Cas d'usage
Pourquoi cette feature est utile?

## Solution proposée
Comment devrait-elle fonctionner?

## Solutions alternatives
D'autres approches possibles?
```

## Pull Request Template

```markdown
## Description
Décrire les changements

## Type de PR
- [ ] Bug fix
- [ ] Feature
- [ ] Documentation

## Related Issues
Fixes #123

## Tests
- [ ] Tests ajoutés
- [ ] Tests existants passent
- [ ] Testé localement

## Checklist
- [ ] Code suit les conventions
- [ ] Documentation mise à jour
- [ ] Pas de breaking changes
```
